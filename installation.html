Installation¶

Requirements¶

Pyscholar required to have the following packages installed:

  • Python >=2.7
  • Networkx
  • Matplotlib

Note

Pyscholar does not support Python 3.x

Step-by-Step Installation Guide¶

Before installing Pyscholar you must get your API Key in the following link.

Once you have the API key. We continue with the installation of Pyscholar.

  1. Clone the repository first as follows:
$ git clone https://github.com/rfabila/Pyscholar.git
  1. Then you must run setup.py and enter your API KEY
$ python setup.py
$ Your scopus api key:(Here you enter your API Key)
  1. Finally we will check the installation for this first navigate to the installation directory then execute python and finally we import the package Pyscholar:
$ cd ~/Pyscholar/src
$ python
Python 2.7.6 (default, Jun 22 2015, 17:58:13)
[GCC 4.8.2] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>>

>>> import pyscholar

If the import of pyscholar not failed then the installation was successful.
